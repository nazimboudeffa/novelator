# Novelator

A portable text editor to write your novels based on [Left](https://github.com/hundredrabbits/Left)

## Install & Run

```
git clone https://github.com/nazimboudeffa/novelator.git
cd novelator/app/
npm install
npm start
```

## Build

```
git clone https://github.com/nazimboudeffa/novelator.git
cd novelator/app/
npm run build_win
```
